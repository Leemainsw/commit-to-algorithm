# 실패율 구하기

슈퍼 게임 개발자 오렐리는 큰 고민에 빠졌다.  
그녀가 만든 **프랜즈 오천성**이 대성공을 거뒀지만, 요즘 신규 사용자의 수가 급감한 것이다.  
원인은 **신규 사용자와 기존 사용자 사이의 스테이지 차이**가 너무 큰 것이었다.

이 문제를 해결하기 위해 오렐리는 **동적으로 게임 시간을 늘려 난이도를 조절**하기로 했다.  
대부분의 로직은 쉽게 구현했지만, **실패율을 구하는 부분**에서 막히고 말았다.

오렐리를 위해 **실패율을 구하는 코드를 완성하라.**

---

## 실패율 정의

> **실패율 = 스테이지에 도달했으나 아직 클리어하지 못한 플레이어 수 / 스테이지에 도달한 플레이어 수**

---

## 문제 설명

- 전체 스테이지의 개수 `N`
- 게임을 이용하는 사용자가 **현재 멈춰있는 스테이지 번호**가 담긴 배열 `stages`

위 정보가 주어질 때,  
**실패율이 높은 스테이지부터 내림차순**으로 스테이지 번호를 담은 배열을 반환하도록  
`solution` 함수를 완성하시오.

---

## 제한사항

- 스테이지의 개수 `N`은 **1 이상 500 이하**의 자연수
- `stages`의 길이는 **1 이상 200,000 이하**
- `stages`에는 **1 이상 N + 1 이하**의 자연수가 담겨 있음
- 각 숫자는 사용자가 **현재 도전 중인 스테이지 번호**
- `N + 1`은 **마지막 스테이지(N번)를 모두 클리어한 사용자**
- 실패율이 같은 스테이지가 있다면 **스테이지 번호가 작은 것부터**
- 스테이지에 도달한 유저가 없는 경우 **실패율은 0**

---

## 입출력 예

| N | stages | result |
|---|---|---|
| 5 | [2, 1, 2, 6, 2, 4, 3, 3] | [3, 4, 2, 1, 5] |
| 4 | [4, 4, 4, 4, 4] | [4, 1, 2, 3] |

---

## 입출력 예 설명

### 예제 1

- **1번 스테이지**
  - 도전한 유저: 8명
  - 클리어 못한 유저: 1명  
  → 실패율 = `1 / 8`

- **2번 스테이지**
  - 도전한 유저: 7명
  - 클리어 못한 유저: 3명  
  → 실패율 = `3 / 7`

- **3번 스테이지**
  - 실패율 = `2 / 4`

- **4번 스테이지**
  - 실패율 = `1 / 2`

- **5번 스테이지**
  - 실패율 = `0 / 1`

실패율 내림차순 정렬 결과:

```text
[3, 4, 2, 1, 5]
